# financial_accounting — 프로젝트 가이드

## 프로젝트 개요

미국 가상자산 회계기준(ASC 350-60)과 국제회계기준(IFRS)을 비교 분석하고, 4개 조사 질문에 대한 근거 기반 분석 보고서를 생성하는 리서치 프로젝트.

## 하네스: 가상자산 회계기준 리서치

**목표:** ASC 350-60과 IFRS 비교 분석 보고서 자동 생성

**트리거:** 가상자산 회계 분석, ASC 350-60 보고서, IFRS 비교 분석, 재무 영향 분석, 주가 영향 분석, 보고서 수정·재실행·업데이트 요청 시 `crypto-accounting-orchestrator` 스킬을 사용하라. 단순 질문은 직접 응답 가능.
